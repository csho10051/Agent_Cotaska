## ステータス遷移

```text
[新規作成] → todo
               │
               ├──→ doing ←──→ blocked
               │      │
               └──────┴──→ done
```

- todo → doing, done
- doing → blocked, done, todo
- blocked → doing, todo
- done → todo（完了取消）

---

## 備考

- クイック追加時の初期値：status='todo', priority='medium', progress=0, delete_flag=0
- list_id が NULL の場合、リストなしに表示
- parent_id が NULL の場合は親を持たないルートタスクとして扱う
- 親子関係は最大5階層までを正式対応範囲とし、サブタスクを親に持つタスクも5階層以内であれば有効な階層として扱う（CHG-045）
- progress_status はチェックボックス状態と双方向同期する
- 子タスクを持つ親タスクは、5階層以内の子孫タスク群から進捗を再計算する
- 完了取消時は completed_at を NULL に戻す
- 削除（ゴミ箱移動）は delete_flag=1 + deleted_at 設定。完全削除は物理DELETE
