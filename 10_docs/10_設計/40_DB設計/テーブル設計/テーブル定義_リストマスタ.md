# テーブル定義 - リストマスタ

## 基本情報

| 項目 | 値 |
|------|-----|
| 論理名 | リストマスタ |
| 物理名 | mst_list |
| 用途 | マスタ |
| 概要 | タスクを分類・整理するためのグループ単位を管理する |

---

## カラム定義

| # | 論理名 | 物理名 | 型 | NOT NULL | デフォルト | 備考 |
|---|--------|--------|-----|:---:|-----------|------|
| 1 | ID | id | INTEGER | ○ | 自動採番 | PK |
| 2 | リスト名 | name | TEXT | ○ | | 1〜100文字 |
| 3 | 表示色 | color | TEXT | | | #RRGGBB形式 |
| 4 | 表示順 | sort_order | INTEGER | ○ | 0 | ナビパネルでの並び順 |
| 5 | 作成日時 | created_at | TEXT | ○ | | ISO 8601 |
| 6 | 更新日時 | updated_at | TEXT | ○ | | ISO 8601 |

---

| 制約名 | 種別 | 対象カラム | 備考 |
|--------|------|-----------|------|

## 特殊リスト（テーブル外）

以下はシステム固定のビューであり、mst_list テーブルには格納しない。

| 名称 | 抽出条件 |
|------|----------|
| 受信トレイ | trx_task.list_id IS NULL |
| 今日 | trx_task.due_date = 今日 |
| 次の7日間 | trx_task.due_date ≤ 今日+7 |
| 完了 | trx_task.status = 'done' |
| ゴミ箱 | trx_task.delete_flag = 1 |

---

## 備考

- 同名リストは許容する（ID で区別）
- リスト削除時、所属タスクの list_id を NULL に変更（受信トレイへ移動）
- color が NULL の場合はUI側でデフォルト色を適用
