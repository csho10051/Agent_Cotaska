# テーブル定義 - タグマスタ

## 基本情報

| 項目 | 値 |
|------|-----|
| 論理名 | タグマスタ |
| 物理名 | mst_tag |
| 用途 | マスタ |
| 概要 | タスクに付与するタグ（ラベル）のマスタデータを管理する |

---

## カラム定義

| # | 論理名 | 物理名 | 型 | NOT NULL | デフォルト | 備考 |
|---|--------|--------|-----|:---:|-----------|------|
| 1 | ID | id | INTEGER | ○ | 自動採番 | PK |
| 2 | タグ名 | name | TEXT | ○ | | 一意 |
| 3 | 表示色 | color | TEXT | | | #RRGGBB形式 |
| 4 | 作成日時 | created_at | TEXT | ○ | | ISO 8601 |
| 5 | 更新日時 | updated_at | TEXT | ○ | | ISO 8601 |

---

## 制約

| 制約名 | 種別 | 対象カラム | 備考 |
|--------|------|-----------|------|
| pk_mst_tag | PRIMARY KEY | id | |
| uq_mst_tag_name | UNIQUE | name | タグ名の重複を防止 |

---

## インデックス

なし（UNIQUE 制約が暗黙的にインデックスを生成）

---

## 備考

- タグ名は一意制約により重複を防止する
- ナビパネルの「タグ」セクションに一覧表示
- タスクとの関連は trx_task_tag で管理（多対多）
- 初期スコープは次段階（初期リリース後に実装）
