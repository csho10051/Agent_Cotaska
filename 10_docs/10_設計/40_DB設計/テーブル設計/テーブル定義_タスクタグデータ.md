# テーブル定義 - タスクタグデータ

## 基本情報

| 項目 | 値 |
|------|-----|
| 論理名 | タスクタグデータ |
| 物理名 | trx_task_tag |
| 用途 | トランザクション（関連付け） |
| 概要 | タスクとタグの多対多の関連を管理する中間テーブル |

---

## カラム定義

| # | 論理名 | 物理名 | 型 | NOT NULL | デフォルト | 備考 |
|---|--------|--------|-----|:---:|-----------|------|
| 1 | ID | id | INTEGER | ○ | 自動採番 | PK |
| 2 | タスクID | task_id | INTEGER | ○ | | FK → trx_task.id |
| 3 | タグID | tag_id | INTEGER | ○ | | FK → mst_tag.id |
| 4 | 作成日時 | created_at | TEXT | ○ | | ISO 8601 |

---

## 制約

| 制約名 | 種別 | 対象カラム | 備考 |
|--------|------|-----------|------|
| pk_trx_task_tag | PRIMARY KEY | id | |
| fk_trx_task_tag_task_id | FOREIGN KEY | task_id | → trx_task(id) ON DELETE CASCADE |
| fk_trx_task_tag_tag_id | FOREIGN KEY | tag_id | → mst_tag(id) ON DELETE CASCADE |
| uq_trx_task_tag_pair | UNIQUE | task_id, tag_id | 同一タスクに同じタグの重複付与を防止 |

---

## インデックス

| インデックス名 | 対象カラム | 備考 |
|---------------|-----------|------|
| ix_trx_task_tag_task_id | task_id | タスクに紐づくタグ一覧の取得 |
| ix_trx_task_tag_tag_id | tag_id | タグに紐づくタスク一覧の取得 |

---

## 備考

- タスク削除（物理DELETE）時は CASCADE で自動削除
- タグ削除時も CASCADE で関連レコードを自動削除
- 初期スコープは次段階（タグ機能と同時に実装）
