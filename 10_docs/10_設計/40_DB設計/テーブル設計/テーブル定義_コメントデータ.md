# テーブル定義 - コメントデータ

## 基本情報

| 項目 | 値 |
|------|-----|
| 論理名 | コメントデータ |
| 物理名 | trx_comment |
| 用途 | トランザクション |
| 概要 | タスクに紐づく作業メモ・経緯のコメントを管理する |

---

## カラム定義

| # | 論理名 | 物理名 | 型 | NOT NULL | デフォルト | 備考 |
|---|--------|--------|-----|:---:|-----------|------|
| 1 | ID | id | INTEGER | ○ | 自動採番 | PK |
| 2 | タスクID | task_id | INTEGER | ○ | | FK → trx_task.id |
| 3 | コメント内容 | content | TEXT | ○ | | コメント本文 |
| 4 | 作成日時 | created_at | TEXT | ○ | | ISO 8601 |

---

## 制約

| 制約名 | 種別 | 対象カラム | 備考 |
|--------|------|-----------|------|
| pk_trx_comment | PRIMARY KEY | id | |
| fk_trx_comment_task_id | FOREIGN KEY | task_id | → trx_task(id) ON DELETE CASCADE |

---

## インデックス

| インデックス名 | 対象カラム | 備考 |
|---------------|-----------|------|
| ix_trx_comment_task_id | task_id | タスクに紐づくコメント一覧の取得 |

---

## 備考

- 詳細ペインのコメントセクションに時系列で表示（created_at 昇順）
- Enter キーで投稿
- タスクの物理削除時は CASCADE で自動削除
- 初期スコープは次段階
