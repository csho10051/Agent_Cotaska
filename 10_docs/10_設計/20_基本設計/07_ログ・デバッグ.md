# 基本設計 - ログ機能・デバッグ

## 1. 概要

CoTasker で発生するログを3つのカテゴリに分類し、それぞれの目的・仕様を定義する。

| ログ種別 | 対象者 | 目的 | 記録先 |
|---------|--------|------|--------|
| **監査ログ** | ユーザー | タスク操作履歴の追跡・表示 | DB + UI |
| **デバッグログ** | 開発者 | IPC/DB/エラーのトレース | console + ファイル |
| **実行ログ** | 開発者 | アプリのシステムヘルス記録 | ファイル |

---

## 2. ユーザー向け監査ログ

### 2.1 目的

- タスク操作の履歴を追跡可能にする
- 詳細ペインで操作履歴を視覚化
- 将来の AI 連携で変更パターン分析に利用

### 2.2 記録範囲

| 操作 | action | 記録内容（details） |
|------|--------|---------------------------|
| タスク作成 | `created` | — |
| タスク編集 | `updated` | `field: old → new` (変更項目のみ) |
| 完了 | `completed` | — |
| 完了取消 | `reopened` | — |
| ゴミ箱移動 | `trashed` | — |
| 復元 | `restored` | — |
| リスト変更 | `moved` | `'旧リスト' → '新リスト'` |
| 進捗更新 | `progress_changed` | `30 → 70` |
| コメント追加 | `commented` | コメント文字数 |
| 並び順変更 | `reordered` | — |

### 2.3 DB スキーマ

```sql
CREATE TABLE tbl_task_history (
  id INTEGER PRIMARY KEY AUTOINCREMENT,
  task_id INTEGER NOT NULL,
  action TEXT NOT NULL,
  details TEXT,
  created_at TEXT NOT NULL,
  FOREIGN KEY (task_id) REFERENCES tasks(id)
);

CREATE INDEX idx_task_history_task_id ON tbl_task_history(task_id);
```

### 2.4 DBサービス層

```javascript
// src/main/db.js 内

/**
 * 監査ログを記録
 * @param {number} taskId
 * @param {string} action - 操作種別 (created, updated, moved など)
 * @param {string} [details] - 変更詳細
 */
function addHistory(taskId, action, details = null) {
  const now = new Date().toISOString();
  const sql = `
    INSERT INTO tbl_task_history (task_id, action, details, created_at)
    VALUES (?, ?, ?, ?)
  `;
  return db.run(sql, [taskId, action, details, now]);
}

/**
 * タスクの履歴一覧を取得（新しい順）
 * @param {number} taskId
 * @returns {object[]}
 */
function getHistoriesByTaskId(taskId) {
  const sql = `
    SELECT id, action, details, created_at
    FROM tbl_task_history
    WHERE task_id = ?
    ORDER BY created_at DESC
  `;
  return db.all(sql, [taskId]);
}

/**
 * タスク削除時に関連履歴を削除
 * @param {number} taskId
 */
function deleteHistoriesByTaskId(taskId) {
  const sql = `DELETE FROM tbl_task_history WHERE task_id = ?`;
  return db.run(sql, [taskId]);
}
```

## 3. 開発者向けデバッグログ

### 3.1 目的

- IPC通信、DB操作、エラーをトレース
- 開発時のトラブルシューティング効率向上
- デバッグ時の情報源

### 3.2 記録対象

| カテゴリ | 内容 |
|---------|------|
| **IPC通信** | リクエスト/レスポンス、パラメータ |
| **DB操作** | SQL実行、返却行数、エラー |
| **React** | コンポーネント初期化、更新、マウント |
| **エラー** | catchしたエラーのスタックトレース |

### 3.3 ログレベル

| レベル | 用途 | 出力例 |
|--------|------|--------|
| `DEBUG` | 詳細情報 | 変数値、フロー制御 |
| `INFO` | 重要な実行履歴 | IPC呼び出し、DB成功 |
| `WARN` | 警告 | タイムアウト、deprecated API |
| `ERROR` | エラー | 例外キャッチ、スタックトレース |

### 3.4 ファイル構成

**ファイル**: `src/main/logger.js`

```javascript
const fs = require('fs');
const path = require('path');

class DebugLogger {
  constructor() {
    this.logDir = path.join(process.env.HOME || process.env.USERPROFILE, '.cotasker', 'logs');
    this.enabled = process.env.NODE_ENV === 'development';
    this.logFile = null;
    
    if (this.enabled) {
      this._ensureLogDir();
      this._openLogFile();
    }
  }

  _ensureLogDir() {
    if (!fs.existsSync(this.logDir)) {
      fs.mkdirSync(this.logDir, { recursive: true });
    }
  }

  _openLogFile() {
    const today = new Date().toISOString().slice(0, 10);
    const filename = `debug-${today}.log`;
    const filepath = path.join(this.logDir, filename);
    
    // append モードでファイルを開く
    this.logFile = fs.createWriteStream(filepath, { flags: 'a' });
  }

  _format(level, message, context = null) {
    const timestamp = new Date().toISOString();
    let output = `[${timestamp}] [${level}] ${message}`;
    if (context) {
      output += ` | ${JSON.stringify(context)}`;
    }
    return output;
  }

  debug(message, context = null) {
    if (!this.enabled) return;
    const msg = this._format('DEBUG', message, context);
    console.log(msg);
    if (this.logFile) this.logFile.write(msg + '\n');
  }

  info(message, context = null) {
    if (!this.enabled) return;
    const msg = this._format('INFO', message, context);
    console.log(msg);
    if (this.logFile) this.logFile.write(msg + '\n');
  }

  warn(message, context = null) {
    if (!this.enabled) return;
    const msg = this._format('WARN', message, context);
    console.warn(msg);
    if (this.logFile) this.logFile.write(msg + '\n');
  }

  error(message, error = null) {
    if (!this.enabled) return;
    let output = `[${new Date().toISOString()}] [ERROR] ${message}`;
    if (error && error.stack) {
      output += `\n${error.stack}`;
    }
    console.error(output);
    if (this.logFile) this.logFile.write(output + '\n');
  }

  destroy() {
    if (this.logFile) {
      this.logFile.end();
    }
  }
}

module.exports = new DebugLogger();
```

### 3.5 使用例

```javascript
// src/main/main.js 内
const logger = require('./logger');

ipcMain.handle('tasks:add', async (_e, task) => {
  logger.info('IPC: tasks:add', { title: task.title });
  try {
    const result = db.addTask(task);
    logger.debug('DB insert success', { id: result.id, rows: 1 });
    return result;
  } catch (err) {
    logger.error('tasks:add failed', err);
    return { ok: false, error: err.message };
  }
});

// Renderer側 (console.log でOK)

```

## 4. アプリ実行ログ

### 4.1 目的

- アプリのシステムヘルスを記録
- クラッシュ時の分析用エビデンス
- 不具合発生時の原因調査

### 4.2 記録対象

| イベント | 記録内容 |
|---------|---------|
| **起動** | 起動時刻、Node版、Electron版、起動引数 |
| **DB初期化** | 初期化完了、テーブル数、インデックス数 |
| **Vite dev server 起動** | Vite版、ポート番号、準備完了 |
| **重大エラー** | エラーメッセージ、スタックトレース、発生箇所 |
| **終了** | 終了時刻、セッション継続時間 |
| **IPC/DB タイムアウト** | 処理内容、タイムアウト時刻、閾値 |

### 4.3 ファイル構成

**ファイル**: `src/main/appLogger.js`

```javascript
const fs = require('fs');
const path = require('path');

class AppLogger {
  constructor() {
    this.logDir = path.join(process.env.HOME || process.env.USERPROFILE, '.cotasker', 'logs');
    this.logFile = null;
    this.startTime = null;
    
    this._ensureLogDir();
    this._openLogFile();
  }

  _ensureLogDir() {
    if (!fs.existsSync(this.logDir)) {
      fs.mkdirSync(this.logDir, { recursive: true });
    }
  }

  _openLogFile() {
    const today = new Date().toISOString().slice(0, 10);
    const filename = `app-${today}.log`;
    const filepath = path.join(this.logDir, filename);
    
    this.logFile = fs.createWriteStream(filepath, { flags: 'a' });
  }

  _write(level, message, data = null) {
    const timestamp = new Date().toISOString();
    let output = `[${timestamp}] [${level}] ${message}`;
    if (data) {
      output += ` | ${JSON.stringify(data)}`;
    }
    output += '\n';
    
    if (this.logFile) {
      this.logFile.write(output);
    }
  }

  logStartup(metadata) {
    this.startTime = Date.now();
    this._write('INFO', 'App startup', {
      version: metadata.version,
      nodeVersion: process.versions.node,
      electronVersion: metadata.electronVersion,
      platform: process.platform,
    });
  }

  logDbInitialization(stats) {
    this._write('INFO', 'Database initialized', {
      tables: stats.tableCount,
      indexes: stats.indexCount,
      duration: stats.duration,
    });
  }

  logShutdown() {
    const duration = this.startTime ? (Date.now() - this.startTime) / 1000 : 0;
    this._write('INFO', 'App shutdown', {
      sessionTime: `${Math.floor(duration)}s`,
    });
  }

  logError(errorMsg, stackTrace = null) {
    let output = `[${new Date().toISOString()}] [ERROR] ${errorMsg}`;
    if (stackTrace) {
      output += `\n${stackTrace}`;
    }
    output += '\n';
    
    if (this.logFile) {
      this.logFile.write(output);
    }
  }

  logWarning(warnMsg, context = null) {
    this._write('WARN', warnMsg, context);
  }

  destroy() {
    if (this.logFile) {
      this.logFile.end();
    }
  }
}

module.exports = new AppLogger();
```

### 4.4 使用例

```javascript
// src/main/main.js 内

// アプリ起動
app.on('ready', async () => {
  appLogger.logStartup({
    version: '0.1.0',
    electronVersion: app.getVersion(),
  });
  
  try {
    await db.initialize();
    appLogger.logDbInitialization({
      tableCount: 6,
      indexCount: 10,
      duration: 145,
    });
  } catch (err) {
    appLogger.logError('DB initialization failed', err.stack);
  }
  
  // ... 以降の処理

// アプリ終了
app.on('window-all-closed', () => {
  appLogger.logShutdown();
  app.quit();
});

// 未キャッチエラー
process.on('uncaughtException', (err) => {
  appLogger.logError('Uncaught exception', err.stack);
});
```

### 4.5 ファイル管理

- **格納先**: `~/.cotasker/logs/`
- **ファイル形式**: `app-YYYYMMDD.log`
- **ローテーション**: 日単位（自動）
- **サイズチェック**: 10MB 到達時に新規作成
- **保持期間**: 最新30日分
- **クリーンアップ**: アプリ起動時に古いファイルを削除

```javascript
// ファイルローテーション・クリーンアップロジック
function cleanupOldLogs() {
  const thirtyDaysAgo = new Date();
  thirtyDaysAgo.setDate(thirtyDaysAgo.getDate() - 30);
  
  fs.readdirSync(this.logDir).forEach(file => {
    const filepath = path.join(this.logDir, file);
    const stat = fs.statSync(filepath);
    
    if (stat.mtime < thirtyDaysAgo) {
      fs.unlinkSync(filepath);
    }
  });
}
```
