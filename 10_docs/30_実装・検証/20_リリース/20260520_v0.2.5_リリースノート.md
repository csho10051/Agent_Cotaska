# Cotaska v0.2.5 リリースノート

## リリース日

2026-05-20

## 概要

今日 / 明日 / 次の7日間ビューにおける保留タスクの表示を改善しました。
保留中のタスクが通常の日付セクションに混在しないようにしつつ、日付ビュー内の保留セクションでも確認できるようにしています。

## 変更内容

- 今日 / 明日 / 次の7日間ビューでは、`progress_status = 保留` のタスクを通常の日付セクションから分離。
- 日付ビュー内に保留セクションを表示し、対象日付範囲内の保留タスクを確認可能に変更。
- 遅延している保留タスクも、今日ビューでは今日以前、明日ビューでは明日以前の保留として表示。
- 左ナビの「保留」ビューでは、従来どおり全保留タスクを確認可能。
- 今日 / 明日 / 次の7日間の件数バッジには、保留タスクを含めない仕様を維持。

## 検証

- `npm run build` 成功。
- Node 24.14.0 環境でビルド確認。
