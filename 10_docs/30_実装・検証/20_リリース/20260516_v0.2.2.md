# Cotaska v0.2.2 リリースノート

## 概要

Cotaska v0.2.2 は、メインペインのクイック追加バーでタスクを入力するときに、日付やリスト指定を自然に入力できるようにした機能改善リリースです。

このリリースでは、タスク名に含まれる日付表現を期日として扱い、検出した日付を入力中にチップで強調表示します。また、末尾に `~` を入力したときに既存リスト候補を表示し、選択したリストへ直接タスクを追加できるようにしました。

## 主な変更

- クイック追加バーで `今日`、`明日`、`明後日`、`YYYY-MM-DD`、`YYYY/MM/DD`、`M/D`、`M月D日` の日付表現を検出し、タスクの `due_date` に反映するようにしました。
- 検出された日付を日付チップとして入力欄上に表示し、保存予定の期日が分かるようにしました。
- タスク名の末尾に `~` を入力したとき、既存リストの候補を表示するようにしました。
- リスト候補を選択すると、選択したリストがタスクの `list` に保存されるようにしました。
- 日付検出とリスト候補表示は、メインペインのクイック追加バーだけに適用し、詳細ペインのタイトル・本文・サブタスク入力には影響しないようにしました。
- 配布物を Cotaska-Portable 版に一本化し、NSISインストーラ、`.blockmap`、`latest.yml` は生成・アップロードしない方針に変更しました。

## 対応した仕様変更

- `CHG-075` タスク入力時の補正機能
- `CHG-076` Portable版のみのリリース運用

## 配布物

GitHub Releases には、次の配布物をアップロードします。

- `Cotaska-Portable.zip`
- `Cotaska-Portable.zip.sha256`

`Cotaska-Portable.zip` は手動更新用の Portable 版です。
`Cotaska-Portable.zip.sha256` は zip の検証用 SHA-256 ファイルです。

## 検証

- `npm run build`
- `release-all.ps1 -Version "0.2.2"`
