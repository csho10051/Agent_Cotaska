# T-018 手動確認チェックリスト

## 対象
- T-015 タスク進捗ステータス管理

## 実施前
- [ ] アプリを起動する（development）
- [ ] 任意の親タスク1件とサブタスク2件以上を用意する

## 確認項目

### 1. 双方向連動（チェック ↔ progress_status）
- [ ] タスク行のチェックを ON にすると progress_status が「完了」になる
- [ ] タスク行のチェックを OFF にすると progress_status が「仕掛」になる
- [ ] 詳細ペインで progress_status を「完了」にするとチェックが ON になる
- [ ] 詳細ペインで progress_status を「未着/仕掛」にするとチェックが OFF になる

### 2. 親タスク自動推定（最悪値）
- [ ] 子に「未着」が1件でもあると親が「未着」になる
- [ ] 子に「未着」がなく「仕掛」があると親が「仕掛」になる
- [ ] 子がすべて「完了」のとき親が「完了」になる

### 3. ToggleButton フィルタ
- [ ] ナビパネルの「未着」ON/OFFで表示タスクが切り替わる
- [ ] ナビパネルの「仕掛」ON/OFFで表示タスクが切り替わる
- [ ] ナビパネルの「完了」ON/OFFで表示タスクが切り替わる
- [ ] 複数選択時に和集合表示になる

### 4. セクション/バッジ表示
- [ ] メインペインでステータス別セクションが表示される
- [ ] 各タスク行に進捗バッジ（未着/仕掛/完了）が表示される
- [ ] ナビパネル件数バッジが概ね一致する
