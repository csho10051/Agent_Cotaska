# CHG-073 バックアップzip形式化

## 1. 概要

バックアップと復元機能のバックアップ成果物を、フォルダコピー方式からzipファイル方式へ変更する。

## 2. 背景

フォルダ形式のバックアップは利用者が移動・保管する際に構成を崩しやすい。バックアップを1ファイル化し、復元時にCotaskaバックアップとして検証できるようにすることで、扱いやすさと安全性を高める。

## 3. 対象範囲

- UI: バックアップ保存先はフォルダ選択のままとし、作成結果は `Cotaska-backup-*.zip` とする
- UI: 復元元選択はバックアップzipファイル選択とする
- main process: `manifest.json` 付きzipバックアップの作成
- main process: zipを一時フォルダへ展開し、構造検証後に復元する
- 互換性: 既存のフォルダ形式バックアップも復元可能なまま残す
