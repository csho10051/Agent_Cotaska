# 仕様変更ファイル

## 変更ID

CHG-004

## 変更内容

DB のタスクデータを Markdown 形式でファイル出力する機能を実装。タスク一覧をセクション分けされたMarkdownドキュメントとして生成し、AI エージェントが読み込める形式で提供する。

## 変更理由

将来的な AI エージェント連携を実現するため、タスク情報を構造化されたテキスト形式で出力する必要がある。Markdown形式により、AI が自然言語処理可能な形式でタスク状況を把握できるようになり、サマリ生成や優先度提案などの高度な機能の基盤となる。

## 影響範囲

- **UI**: 
  - MainPane ヘッダーまたは詳細ペインに「Markdown出力」ボタン追加
  - 出力完了時の確認メッセージ表示
- **API/IPC**: 
  - `ai:exportMarkdown(options?)`
  - options に含まれるフィルター条件（例：未完了のみ、期限内など）
- **ファイルシステム**:
  - 出力先：`Cotaska/00_mgmt/CURRENT_SPRINT.md` または `40_workspace/temp/`
- **既存機能影響**:
  - なし（新規ボタン追加のみ）

## 設計反映先
