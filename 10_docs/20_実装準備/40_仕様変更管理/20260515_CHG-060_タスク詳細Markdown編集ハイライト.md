# 仕様変更ファイル

## 変更ID

CHG-060

## 変更内容

タスク詳細本文の編集モードに、Markdown 記法のシンタックスハイライトを追加する。VSCode の Markdown 編集画面に近い操作感を目標とし、保存データは従来どおりプレーンな Markdown テキストのまま維持する。

## 変更理由

現在の編集モードは通常の textarea 表示であり、`## 見出し` や `**強調**` などの Markdown 記法を書いても編集中には視覚的な区別がない。長いタスク本文を編集するとき、見出しや強調箇所を把握しやすくするため。

## 影響範囲

- UI: タスク詳細本文の編集モード、プレビュー切替、本文入力体験
- DB: なし
- API/IPC: 原則なし
- 既存機能影響: 自動保存、タスク詳細プレビュー、リンク起動、外部アプリ起動、無効タスク表示の動作確認が必要
