# 仕様変更ファイル

## 変更ID

CHG-023

## 変更内容

右詳細ペインの優先度表示・操作方式を変更する。

- 変更前: `N` / `M` / `H` のアルファベット略字をクリックして循環変更（cyclePriority）
- 変更後: `低` / `中` / `高` のプルダウン（`<select>`）で選択

内部データキー（`normal` / `medium` / `high`）・保存形式・ソート順は変更しない。

## 変更理由

現状の N/M/H 表記はユーザーにとって直感的でなく、意味が分かりにくい。
また、クリックで循環変更する方式は誤操作が発生しやすい。
プルダウン方式に統一することで、操作の明確性と日本語UIの一貫性を確保する。

## 影響範囲

- UI:
  - 右詳細ペインの優先度表示エリアを `<span>` クリック循環から `<select>` プルダウンへ変更
- 実装:
  - `20_app/src/renderer/components/DetailPane.jsx`
    - `PRIORITY_ICON` 定数を廃止
    - `cyclePriority` 関数を廃止
    - `d-priority` `<span>` を `<select>` 要素に置換
    - 選択肢: `低`（normal）/ `中`（medium）/ `高`（high）
  - `20_app/src/renderer/components/DetailPane.css`（または同等スタイル）
    - `d-priority` セレクタのスタイル調整（必要時）
- 設計:
  - `10_docs/10_設計/30_画面モック/layout_mock.html`
    - 優先度エリアをプルダウン表示に更新
- 変更なし:
  - `20_app/src/renderer/App.jsx` の `PRIORITY_ORDER` / `sortByPriority`（内部キーは不変）
  - タスクデータ保存形式（`priority: "normal" | "medium" | "high"` は維持）
  - DB スキーマ（変更なし）
