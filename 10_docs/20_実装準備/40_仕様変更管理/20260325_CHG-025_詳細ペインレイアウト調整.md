# 仕様変更ファイル

## 変更ID

CHG-025

## 変更内容

右詳細ペインの以下4箇所のレイアウト・UI変更を行う。

1. **優先度ラベル追加**: ヘッダー行の優先度表示要素（`d-priority`）の直前に「優先度：」のテキストラベルを追加する
2. **進捗ステータス行を左寄せ**: 「進捗ステータス」行（`progress-status-row`）を `justify-content: space-between`（両端揃え）から `justify-content: flex-start`（左寄せ）に変更する
3. **保存インジケーター削除**: ヘッダー行に表示される「保存中...」「保存済み」の表示要素を削除する
4. **×ボタン削除**: ヘッダー行右端の閉じるボタン（`close-btn`）を削除する

## 変更理由

1. 優先度の値（低/中/高）だけでは何の欄かわかりにくいため、ラベルを明示して可読性を上げる
2. 「進捗ステータス」のラベルとプルダウンが左右に離れて表示されており、関連する要素として視覚的にまとまりが必要
3. 自動保存が常時行われているため「保存中...」「保存済み」の都度表示は不要で、UIノイズを減らす
4. 詳細ペインを閉じる操作は別の手段（リスト行クリック等）で代替でき、ヘッダーをシンプルにする
