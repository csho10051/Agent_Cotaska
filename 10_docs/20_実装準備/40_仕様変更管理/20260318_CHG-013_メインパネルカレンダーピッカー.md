# CHG-013：メインパネルのタスク行に日付カレンダーピッカー追加

## 変更概要

メインパネルのタスク行に表示される日付（due_date）にたいしてカーソルを当てると、  
カレンダーウィジェットが表示される。日付をクリックするとタスクの期限が更新される。

## 変更前の動作

- タスク行の日付は表示のみ（テキスト）
- 期限変更は詳細ペインの日付から行う

## 変更後の動作

- タスク行の日付にマウスホバー → 小型のカレンダーピッカーがポップアップ
- カレンダー上の日付をクリック → タスクの `due_date` が即座に更新
- 日付クリア機能も備える（「×」ボタン等）
- ESC キーまたは外側クリックでポップアップを閉じる

## 実装個所

- `MainPane.jsx` → タスク行の日付表示部分にホバー検知・ピッカー表示を追加
- `App.jsx` → カレンダー選択後に `handleSetTaskDue()` コールバックを追加
- 共通部品 `DueDatePopover.jsx` を追加し、日付変更 / クリア / ESC / 外側クリックでの終了に対応
