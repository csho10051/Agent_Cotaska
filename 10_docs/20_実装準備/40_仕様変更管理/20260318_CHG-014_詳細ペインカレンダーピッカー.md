# CHG-014：詳細ペインの日付をカレンダーピッカーで編集

## 変更概要

詳細ペイン上部の日付（📅）をクリック可能にし、カレンダーピッカーで期限を変更できるようにする。

## 変更前の動作

- 詳細ペインの日付は表示のみ（テキスト）
- 期限変更はできない、または別途モーダルが開く

## 変更後の動作

- 詳細ペイン上部の日付部分（📅 YYYY/MM/DD）をクリック → カレンダーピッカーが表示
- カレンダー上の日付をクリック → タスクの `due_date` が即座に更新
- 日付クリア機能も備える（「×」ボタン等）
- ESC キーまたは外側クリックでピッカーを閉じる

## 実装個所

- `DetailPane.jsx` → 日付表示部分をクリックハンドラ + カレンダーコンポーネント表示に変更
- `App.jsx` → 詳細ペインからカレンダー選択時に `persist()` で即座に保存
- `DueDatePopover.jsx` を共通利用し、日付クリア / ESC / 外側クリックでの終了に対応
