# CHG-077 起動ロード画面

## 変更ID

CHG-077

## 変更内容

Cotaska.exe 起動直後からメイン画面表示までの間、黒い空白ウィンドウではなくロード画面を表示する。
React の JavaScript ロード前と、初回 `tasks:getAll` 完了前の両方をロード画面の対象にする。
main/renderer 間で起動進捗を通知し、ロード画面には進捗率と現在処理中の内容を表示する。

## 変更理由

Cotaska.exe をダブルクリックしてからメイン画面が表示されるまでの間、黒いウィンドウがしばらく表示される。
利用者からはアプリが固まっているように見えるため、起動中であることが分かるロード画面を表示したい。

## 影響範囲

- UI: 起動時の初期表示、初回タスク読み込み中の表示、進捗率と処理内容の表示
- DB: なし
- API/IPC: `startup:getProgress` と `startup:progress` を追加し、起動進捗を renderer へ渡す
- 既存機能影響: 起動後のタスク一覧、ナビ、詳細ペインの通常操作に影響しないことを確認する
