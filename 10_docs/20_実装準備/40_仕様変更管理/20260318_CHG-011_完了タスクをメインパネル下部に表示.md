# CHG-011：完了タスクをメインパネル下部に表示

## 変更概要

タスクを完了にした際、メインパネルのリストから消えるのではなく、  
リスト下部の **[完了]** セクションに折りたたみ表示するよう変更する。

参考UI: TickTick（今日ビュー）

## 変更前の動作

- タスクを完了にすると `status = "done"` になり、各ビューのフィルタで除外される
- 完了済みタスクは「完了」専用ビューにのみ表示される

## 変更後の動作

- タスクを完了にしても現在のビューのリストに残る
- リスト下部に **「✅ 完了 (n)」** セクションヘッダーが表示される
- セクションはクリックで展開/折りたたみ可能（デフォルト: 展開）
- 完了セクション内のタスクに対してチェック取消・クリック（詳細表示）が可能
- ゴミ箱ビュー・完了専用ビューでは完了セクションを表示しない

## 対象ビュー

| ビュー | 完了セクションの対象タスク |
|--------|--------------------------|
| 今日 | `due_date <= 今日` の完了タスク |
| 次の7日間 | `due_date <= 今日+7日` の完了タスク |
| 受信トレイ / リストなし | `list = null` の完了タスク |
| リスト（個別） | そのリストに属する完了タスク |
| タグ（個別） | そのタグを持つ完了タスク |

## 変更対象ファイル

- `20_app/src/renderer/App.jsx` — `completedSectionTasks` 計算ロジック追加
- `20_app/src/renderer/components/MainPane.jsx` — 完了セクション表示追加
