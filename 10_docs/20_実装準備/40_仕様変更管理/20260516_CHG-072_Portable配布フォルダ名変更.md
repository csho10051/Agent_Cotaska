# CHG-072 Portable配布フォルダ名変更

## 概要

portable版のリリースフォルダ名を `Cotaska-dist` から `Cotaska-Portable` に変更する。

## 背景

GitHub Releases へ portable版と NSISインストーラ版を併記する運用になったため、portable版であることが利用者に伝わる名前へ変更する。

## 変更内容

- `20_app/release/Cotaska-Portable/` を portable版の配布フォルダ正本とする。
- `20_app/release/Cotaska-Portable.zip` を GitHub Releases へ添付する portable版配布物とする。
- `release-all.ps1`、`organize-release.ps1`、`sync-task-master-release.ps1` の参照先を更新する。
- アプリ内の配布フォルダ表示とクラウド同期警告文言を更新する。
- リリース運用ルールと AGENTS.md のアップロード方針を更新する。

## 影響範囲

- portable版の配布フォルダ名とZIPファイル名
- リリース生成・正本反映スクリプト
- 設定画面のアプリ情報表示
- 起動時のユーザー案内文言

## 対象外

- NSISインストーラ名
- electron-updater 用の `latest.yml` / `.blockmap` の形式
- 過去リリース記録の履歴名称

## 完了条件

- `release-all.ps1 -Version "0.1.1"` で `Cotaska-Portable/` と `Cotaska-Portable.zip` が生成される。
- ZIP内のルートフォルダが `Cotaska-Portable/` になっている。
- `sync-task-master-release.ps1` が `Cotaska-Portable/` を同期元にできる。
- 運用ルール上の portable版アップロード対象が `Cotaska-Portable.zip` になっている。
