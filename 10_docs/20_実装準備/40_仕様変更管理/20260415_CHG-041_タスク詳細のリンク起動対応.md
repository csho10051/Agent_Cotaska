# 仕様変更ファイル

## 変更ID

CHG-041

## 変更内容

タスク詳細本文に記載されたリンク情報について、Markdown 準拠を前提に以下を実現する。

- URL リンクをクリックしたとき、既定ブラウザで対象ページを開く
- ローカルファイルパスをクリックしたとき、既定アプリで対象ファイルを開く
- フォルダパスをクリックしたとき、エクスプローラで対象フォルダを開く

あわせて、現状発生している「URL クリック時に画面が真っ暗になる」事象を解消する。

---

## 変更理由

- タスク詳細から関連資料・外部情報へ直接遷移できるようにしたい
- Markdown の運用を維持したままリンク操作性を向上したい
- URL クリックで画面が真っ暗になる不具合を解消し、業務利用時の安全性を確保したい

---

## 影響範囲

- UI: タスク詳細本文のリンク表示・クリック挙動
- DB: なし
- API/IPC: renderer から main へのリンク起動 IPC（URL / file / folder 判定を含む）
- 既存機能影響: Markdown プレビュー、既定アプリ起動処理、エラーダイアログ表示
