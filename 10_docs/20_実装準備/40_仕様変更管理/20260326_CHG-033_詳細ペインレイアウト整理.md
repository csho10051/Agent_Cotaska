# 仕様変更ファイル

## 変更ID

CHG-033

## 変更内容

DetailPane（タスク詳細ペイン）のレイアウトを整理し、メタ情報入力欄をヘッダー直下に集約する。

### 現状の問題点

- 入力項目が画面の上部（期限・優先度）と下部（進捗ステータス・リスト）に散らばっている
- リストセレクトがフッターに配置されており、横のスペースが無駄になっている
- タグエリアと進捗ステータスがメモ本文とフッターの間に挟まれ、レイアウトが煩雑

### 変更前のレイアウト（上から下）

```
┌──────────────────────────────────────┐
│ ☐ 期限:未設定  優先度:[低▼] [プレビュー] │  detail-header
├──────────────────────────────────────┤
│ タスク名                              │  detail-body
│ メモを入力...                         │
│                                       │
│              (メモ本文領域)            │
│                                       │
├──────────────────────────────────────┤
│ #tag1 #tag2                           │  tag-area
│ [タグ選択▼] [新しいタグ___] [追加]    │
├──────────────────────────────────────┤
│ 進捗ステータス [未着▼]  完了日時:...  │  progress-status-row
├──────────────────────────────────────┤
│ [リストなし▼]             ID: T-0035  │  detail-footer
└──────────────────────────────────────┘
```

### 変更後のレイアウト（上から下）

```
┌──────────────────────────────────────┐
│ ☐ タスク名_______________[プレビュー]│  detail-header (タイトルをヘッダーに移動)
├──────────────────────────────────────┤
│ 進捗:[未着▼] 期限:未設定 優先度:[低▼]│  detail-meta (row 1)
│ [なし▼] [タグ選択▼][入力][追加]        │  detail-meta (row 2)
│ #tag1 #tag2                           │  detail-meta (row 3: タグチップス)
├──────────────────────────────────────┤
│ メモを入力...                         │  detail-body (メモ本文のみ、flex:1)
│                                       │
│              (メモ本文領域 拡大)       │
│                                       │
│                                       │
├──────────────────────────────────────┤
│ ID: T-0035                            │  detail-footer (IDのみ)
└──────────────────────────────────────┘
```

---

## 変更理由

1. メタ情報の入力欄（期限・優先度・進捗・リスト・タグ）がペインの上部と下部に散らばっており、操作時に視線移動が大きい
2. リストセレクトがフッターにあるが、横にスペースが余っており無駄になっている
3. メタ情報を上部に集約することで、メモ本文領域をより広く確保できる
4. 関連する項目をグルーピングすることで、視覚的なまとまりが出る

## 変更詳細

### 1. detail-header の変更

| 項目 | 変更前 | 変更後 |
|------|--------|--------|
| 内容 | チェックボックス + 期限 + 優先度 + プレビュー | チェックボックス + タイトル入力 + プレビュー |
| 期限・優先度 | ヘッダー内 | detail-meta へ移動 |
| タイトル | detail-body 内 | ヘッダー内に移動（インライン入力） |

### 2. detail-meta の新設

- ヘッダー直下に新しいメタ情報セクションを設置
- 3列グリッド構成で項目をコンパクトに配置

| 行 | 左列 | 中列 | 右列 |
|----|------|------|------|
| Row 1 | 進捗ステータス | 期限（クリックで編集） | 優先度セレクト |
| Row 2 | リストセレクト | タグ選択プルダウン | タグ入力＋追加ボタン |
| Row 3 | タグチップス表示（横幅全体を使用） | | |

#### 折りたたみ仕様

- 「▼ メタ情報」ヘッダーをクリックすると、Row 1〜Row 3 の表示/非表示をトグルする
- 折りたたみ時はヘッダーが「▶ メタ情報」に変わり、Row 1〜Row 3 は非表示になる
- 展開時はヘッダーが「▼ メタ情報」に戻り、Row 1〜Row 3 を表示する
- 初期状態: **展開**
- 状態管理: `DetailPaneBody` 内の `useState` で管理（永続化不要）

### 3. detail-body の変更

| 項目 | 変更前 | 変更後 |
|------|--------|--------|
| 内容 | タイトル入力 + メモ本文 | メモ本文のみ |
| 領域 | タイトル分のスペース消費あり | メモ領域が拡大 |

### 4. progress-status-row の廃止

- 独立セクションを廃止し、detail-meta の Row 2 に統合

### 5. detail-footer の簡素化

| 項目 | 変更前 | 変更後 |
|------|--------|--------|
| 内容 | リストセレクト + タスクID | タスクIDのみ |
| リストセレクト | フッター内 | detail-meta の Row 2 右列へ移動 |

- DB / IPC / ファイル形式: **変更なし**
- 既存機能影響: **なし**（レイアウト変更のみ、機能動作はすべて維持）
