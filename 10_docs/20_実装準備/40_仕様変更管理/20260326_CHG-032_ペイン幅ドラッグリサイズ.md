# 仕様変更ファイル

## 変更ID

CHG-032

## 変更内容

NavPanel（左ナビ）と DetailPane（詳細ペイン）の幅をドラッグ操作で変更可能にする。

- 変更前: NavPanel = 240px 固定、DetailPane = 36% 固定
- 変更後: NavPanel と DetailPane の境界線をドラッグすることで任意の幅に変更可能

### 表示仕様

| 項目 | 内容 |
|------|------|
| リサイズ対象 | NavPanel（左ナビ）、DetailPane（詳細ペイン）|
| 操作方法 | 各ペイン境界のハンドル（5px 幅）をドラッグ |
| カーソル | `col-resize` |
| NavPanel 幅範囲 | 160px〜480px（初期値 240px）|
| DetailPane 幅範囲 | 280px〜640px（初期値 380px）|
| ハンドル表示 | hover 時にグレーで可視化 |

---

## 変更理由

画面幅や作業内容に応じてペイン幅をユーザが自由に調整できるようにする。  
特に DetailPane の幅が広い方が本文編集しやすい場合や、NavPanel を狭くして一覧を広く使いたい場合に対応する。

---

## 影響範囲

- UI:
  - `App.jsx`: `navWidth` / `detailWidth` state 追加、ドラッグ処理追加、resize handle 挿入
  - `App.css`: `.nav-panel` / `.detail-pane` の固定幅を `width: 100%` に変更、`.resize-handle` スタイル追加
- DB / IPC / ファイル形式: **変更なし**
- 既存機能影響: **なし**（レイアウト変更のみ）

---

## 実装タスク分解

1. `App.css` に `.resize-handle` スタイルを追加する
2. `App.css` の `.nav-panel` / `.detail-pane` 幅を `width: 100%` に変更する
3. `App.jsx` に `navWidth` / `detailWidth` state と `resizeDragRef` を追加する
4. `App.jsx` に `mousemove` / `mouseup` グローバルリスナーを追加する
5. `App.jsx` の JSX に wrapper div と resize handle を挿入する
6. 動作確認（ドラッグリサイズ、幅制限、各ペインの表示崩れなし）
