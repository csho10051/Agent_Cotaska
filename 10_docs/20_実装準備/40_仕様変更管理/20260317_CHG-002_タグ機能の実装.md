# 仕様変更ファイル

## 変更ID

CHG-002

## 変更内容

タスクに対してタグ（ラベル）を付与・削除できる機能を実装。タグはマスタデータとして管理され、ナビパネルに「タグ」セクションを追加して、タグによるフィルタリングを可能にする。

## 変更理由

タスクの分類・検索効率を向上させるため、タグ機能を導入する。複数のリストに属さないタスクの柔軟な分類を実現し、ユーザーの作業効率を高める。

## 影響範囲

- **UI**: 
  - ナビパネルに「タグ」セクション追加（タグ一覧と「+」ボタン）
  - 右クリックメニューの「タグ設定」を有効化し、タグ選択サブメニュー表示
  - 詳細ペインに「タグ」セクション追加（タグ表示・付与・削除UI）
- **DB**: 
  - `mst_tag` テーブル（タグマスタ）新規作成
  - `trx_task_tag` テーブル（タスク-タグ関連、多対多）新規作成
  - タスク表示時にタグ一覧を結合取得
- **API/IPC**: 
  - `tags:getAll` / `tags:add` / `tags:update` / `tags:delete`
  - `tasks:addTag` / `tasks:removeTag` / `tasks:getByTag`
- **既存機能影響**: 
  - メインペインのタスク表示に「タグ」カラム追加（可視性は設定で調整可能）
  - トランザクションで `tasks:getAll` が `trx_task_tag` も返すように拡張
