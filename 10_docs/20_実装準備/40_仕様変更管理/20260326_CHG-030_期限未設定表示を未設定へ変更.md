# CHG-030: 期限未設定表示を未設定へ変更

## 変更ID

CHG-030

## 変更内容

タスク期限が未設定の場合に表示される文言を `期限未設定` から `未設定` へ変更する。
対象は詳細ペインの期限欄と、メイン画面左上の期限欄の 2 箇所。

## 変更理由

現在の表示 `期限未設定` は冗長であり、UI の視認性と表記統一の観点で改善要求が出た。
`未設定` に統一することでコンパクトかつ一貫性のある表示となる。

## 影響範囲

- UI: 詳細ペイン・メインペインの期限表示文言
- DB: なし
- API/IPC: なし
- 既存機能影響: 期限設定済みタスクの表示には影響しない

## 設計反映先

- 設計書への反映不要（文言のみの変更のため）

## 実装タスク分解

1. CHG-030-01: `DetailPane.jsx` の `"期限未設定"` を `"未設定"` に変更する
2. CHG-030-02: `MainPane.jsx` の `"期限未設定"` を `"未設定"` に変更する
3. CHG-030-03: ビルド確認（`npm run build`）・配布再生成・起動確認を実施する
