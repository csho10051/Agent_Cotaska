# 仕様変更ファイル

## 変更ID

CHG-040

## 変更内容

タスク詳細ペイン右上の「プレビュー」導線を見直し、以下を追加する。

- 既存の「プレビュー」ボタンは虫眼鏡アイコン表示へ変更する
- 余白に「新しいアプリで開く」アイコンボタンを追加する
- 「新しいアプリで開く」押下時は、対象タスク Markdown ファイルを OS の既定アプリで開く

なお、既定アプリ未設定または起動失敗時の通知方法は本 CHG の設計確認で確定する。

---

## 変更理由

- 詳細ペインのプレビュー導線をコンパクトにし、視認性と操作性を上げたい
- Markdown 本文を外部エディタやビューアで確認したい要望がある
- タスク詳細を Cotaska 内だけでなく既定アプリ側でも確認できるようにしたい

---

## 影響範囲

- UI: 詳細ペイン右上のボタン表示、アイコン、ツールチップ、配置
- DB: なし
- API/IPC: renderer から main 経由で既定アプリ起動を行う IPC 追加の可能性あり
- 既存機能影響: 現行プレビュー導線、タスクファイルパス解決、起動失敗時のユーザ通知
