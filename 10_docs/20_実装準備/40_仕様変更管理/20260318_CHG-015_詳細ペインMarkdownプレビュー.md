# CHG-015：詳細ペインの Markdown プレビュー機能

## 変更概要

詳細ペイン上部ヘッダーに「プレビュー」ボタンを追加し、  
メモ欄に入力した Markdown テキストをレンダリング表示できるようにする。

## 変更前の動作

- メモ欄は テキスト編集のみ、プレビューなし
- Markdown 記法の確認にはファイルを開いて別途確認

## 変更後の動作

1. 詳細ペインのヘッダーに「プレビュー」ボタンを追加
2. ボタンクリック → メモ欄の表示が「編集モード」 ↔ 「プレビューモード」で切り替わる
3. プレビューモード：メモの Markdown が HTML にレンダリングされて表示される
4. ボタンテキスト：「プレビュー」or「編集」で現在のモードを示す

## Markdown 対応範囲

- 見出し（# ## ###）
- 段落・改行
- **太字** / *斜体*
- リスト（UL / OL）
- コードブロック（三重バッククォート）
- リンク / 画像（オプション）

内部的には `markdown-it` などのライブラリを使用推奨。

## 実装個所

- `DetailPane.jsx` → ヘッダー右側に「プレビュー」ボタン追加
- `DetailPane.jsx` → state で `previewMode` を管理し、本文エリアの表示切り替え
- Markdown→HTML レンダラ用に NPM ライブラリ追加（package.json 更新）
- `markdown-it` を導入し、見出し / リスト / 強調 / コードブロックのプレビューを提供
