# 仕様変更ファイル

## 変更ID

CHG-005

## 変更内容

外部のAIエージェント（Codex等）がタスク情報を参照・分析し、自動提案機能を提供する連携基盤の構築。初期段階では API 設計と連携インターフェースの定義を行い、具体的な提案機能（サマリ生成、優先度提案）は次段階で実装する。

## 変更理由

AI エージェントの能力を活用してタスク管理の効率化・自動化を実現するために、外部連携を可能にする基盤が必要。まずは Markdown出力、ログデータの整備により、AI が読み込み可能な形式でデータを提供する基礎を整える。

## 影響範囲

- **Architecture**: 
  - 新規モジュール `src/main/aiService.js` 設計（AIエージェント連携の中核）
  - IPC チャンネル拡張（ai:generateSummary, ai:suggestPriorityTasks 等）
- **API/IPC**: 
  - `ai:exportMarkdown()` ← CHG-004 で実装済
  - `ai:generateSummary()` ← 将来
  - `ai:suggestPriorityTasks(limit?)` ← 将来
  - `ai:analyzeOverdueTasks()` ← 将来
- **ファイルシステム・外部連携**:
  - AI エージェント用の入出力ディレクトリ設定
  - API キー・認証情報の安全な管理（環境変数）
  - 外部API呼び出しのタイムアウト・リトライロジック
- **既存機能影響**:
  - ログ記録（既存の task_logs）を AI 分析対象に含める
  - detailed_logs（今後の実装）で細粒度な分析が可能
