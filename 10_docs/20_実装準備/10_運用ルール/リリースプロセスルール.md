# CoTasker リリースプロセスルール

## 1. 目的

本ルールは、CoTasker のリリース作業を再現可能にし、誰が実施しても同じ成果物を作れるようにすることを目的とする。

---

## 2. 配布構成（正本）

配布フォルダは以下を正本とする。

- `20_app/release/CoTasker-0.1.0-dist/`

配布フォルダ直下の見え方は次のとおり。

- `CoTasker.exe`（Go ランチャー）
- `_app/`（Electron 本体一式）

補足:

- DLL や resources は `_app/` 配下に配置する
- ルートを最小構成にし、配布先での誤操作を減らす

---

## 3. 前提条件

### 3.1 ツール

- Windows 10/11
- Node.js v22 系
- npm
- Go 1.21 以上（ランチャービルド用）

### 3.2 事前確認

- `CURRENT_SPRINT.md` が最新状態になっている
- `DECISIONS.md` に必要な設計判断が反映されている
- 既知不具合の状態を確認している（特に BUG-20260323-01）

---

## 4. リリース手順

### Step 1. Renderer ビルド

```powershell
cd "C:\WorkDevelop\Agent_TaskHub\CoTasker\20_app"
npm run build
```

確認:

- `dist/renderer/` が生成される
- ビルドエラーがない

### Step 2. Electron 配布物生成（dir）

```powershell
cd "C:\WorkDevelop\Agent_TaskHub\CoTasker\20_app"
npm run dist:dir
```

`dist:dir` は次のコマンドを実行すること（固定）:

```powershell
electron-builder --dir --win --x64 --publish never
```

確認:

- `release/win-unpacked/` が生成される
- `release/win-unpacked/CoTasker.exe` が存在する

### Step 3. Go ランチャーをビルド

```powershell
cd "C:\WorkDevelop\Agent_TaskHub\CoTasker\20_app\setup\launcher"
PowerShell -ExecutionPolicy Bypass -File build.ps1
```

確認:

- `20_app/setup/launcher/CoTasker.exe` が生成される

### Step 4. 配布フォルダを再構成

```powershell
$releaseDir  = "C:\WorkDevelop\Agent_TaskHub\CoTasker\20_app\release"
$distDir     = Join-Path $releaseDir "CoTasker-0.1.0-dist"
$launcherExe = "C:\WorkDevelop\Agent_TaskHub\CoTasker\20_app\setup\launcher\CoTasker.exe"

# 既存配布フォルダを削除
Remove-Item $distDir -Recurse -Force -ErrorAction SilentlyContinue

# 配布フォルダを作成
$appDir = Join-Path $distDir "_app"
New-Item -ItemType Directory -Path $appDir -Force | Out-Null

# Electron 本体を _app へコピー
Copy-Item (Join-Path $releaseDir "win-unpacked\*") $appDir -Recurse -Force

# ランチャーをルートへ配置
Copy-Item $launcherExe (Join-Path $distDir "CoTasker.exe") -Force
```

### Step 5. 起動確認

```powershell
& "C:\WorkDevelop\Agent_TaskHub\CoTasker\20_app\release\CoTasker-0.1.0-dist\CoTasker.exe"
```

確認:

- CoTasker プロセスが起動する
- ウィンドウが表示される
- 必要に応じて `CoTasker.cmd` で比較起動する

---

## 5. 出荷前チェックリスト

- 配布正本が `CoTasker-0.1.0-dist` である
- ルート直下は `CoTasker.exe` と `_app/` のみ
- `CoTasker.exe` から起動できる
- 主要導線（タスク一覧表示、編集、保存）が動作する

---

## 6. 不具合時の暫定運用

- EXE ランチャー起動に問題が出る場合は、`CoTasker.cmd` を暫定運用として許可する
- 再発時は BUG-20260323-01 に追記し、再現手順とログを残す

---

## 7. 更新ルール

本ドキュメントを更新した場合は、以下を同時に更新する。

- `CURRENT_SPRINT.md`（タスク状態）
- `DECISIONS.md`（設計判断）
- 必要に応じて不具合レポート
