## 検証ポイント

### 1. ユーザー向け監査ログ (T-008-01)

**確認方法**:
1. アプリ起動 → タスです作成
2. タスクを選択 → 詳細ペイン右下に「履歴」セクション表示
3. 以下操作を実施し、大義を記録：
   - [ ] タスク作成 → 「created」ログ
   - [ ] 優先度変更 → 「priority_changed」ログ
   - [ ] 進捗更新 → 「progress_changed」ログ
   - [ ] リスト移動 → 「moved」ログ
   - [ ] タスク完了 → 「completed」ログ
   - [ ] 完了取消 → 「reopened」ログ
   - [ ] ゴミ箱移動 → 「trashed」ログ
   - [ ] 復元 → 「restored」ログ

**期待される UI表示**:
```

─────────────────────

2026-03-17 15:30:40  trashed
2026-03-17 15:30:35  completed

2026-03-17 15:28:00  priority_changed: normal → high
2026-03-17 15:27:00  progress_changed: 0% → 50%
2026-03-17 15:26:00  created
```

### 2. 開発者向けデバッグログ (T-008-02)

**確認方法**:
1. DevTools（F12）の Console タブを開く
2. 各操作後に以下フォーマットのログが出力されることを確認：

**期待ログ出力**:
```
[2026-03-17T14:00:00.123Z] [INFO] IPC: tasks:add | {"title":"新タスク","list_id":null}
[2026-03-17T14:00:00.456Z] [DEBUG] DB insert success | {"id":5}
[2026-03-17T14:00:01.000Z] [INFO] IPC: histories:add | {"taskId":5,"action":"created"}
[2026-03-17T14:00:01.200Z] [DEBUG] history added | {"taskId":5,"action":"created"}

[2026-03-17T14:00:05.500Z] [INFO] IPC: tasks:update | {"id":5}
[2026-03-17T14:00:05.700Z] [DEBUG] DB update success | {"id":5}
[2026-03-17T14:00:05.900Z] [INFO] IPC: histories:add | {"taskId":5,"action":"priority_changed"}
```

**ログレベル確認**:
- `[DEBUG]` - 詳細情報（変数値、IPC応答）
- `[INFO]` - 重要な履歴（IPC呼び出し、DB成功）
- `[WARN]` - 警告（タイムアウト等）
- `[ERROR]` - エラー（例外キャッチ時）

### 3. アプリ実行ログ (T-008-03)

**ファイル検証**:

```powershell
# PowerShell でログファイルを確認
$logDir = "$env:USERPROFILE\.Cotaska\logs"
ls $logDir  # ファイル一覧表示

# 本日分ログを表示（ワイプアウト末尾30行）
Get-Content "$logDir\app-$(Get-Date -Format 'yyyy-MM-dd').log" -Tail 30

# デバッグログも確認可能
Get-Content "$logDir\debug-$(Get-Date -Format 'yyyy-MM-dd').log" -Tail 30
```

**期待されるアプリログ内容**:
```
[2026-03-17T14:00:00.123Z] [INFO] App startup | {"version":"0.1.0","nodeVersion":"22.14.0","electronVersion":"41.0.2","platform":"win32"}
[2026-03-17T14:00:01.200Z] [INFO] Database initialized | {"tables":3,"indexes":2,"duration":145}
[2026-03-17T14:00:02.400Z] [INFO] Vite dev server started | {"port":5173}
[2026-03-17T14:30:00.500Z] [INFO] App shutdown | {"sessionTime":"30m 0s"}
```

**ファイル管理確認**:
- [ ] `~/.Cotaska/logs/app-YYYY-MM-DD.log` に毎日の起動・終了ログが記録
- [ ] `~/.Cotaska/logs/debug-YYYY-MM-DD.log` に本日のデバッグログが蓄積
- [ ] 30日以上前のファイルは自動削除（cleanup実装確認）

## トラブルシューティング

### ログファイルが作成されない場合
```powershell
# ディレクトリのパーミッション確認
ls $env:USERPROFILE\.Cotaska -Force
# 手動作成
mkdir "$env:USERPROFILE\.Cotaska\logs" -Force
```

### DevTools に日本語が文字化けている
- Electron側の設定で UTF-8 を確認
- ログレベルやメッセージは英字のため影響少

### 履歴が表示されない
1. ブラウザコンソールでエラーを確認
2. DB テーブル確認: `sqlite> SELECT * FROM tbl_task_history;`
3. IPC通信を DevTools で確認
