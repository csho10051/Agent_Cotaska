# DECISIONS

## 2026-05-03 CHG-045 設計書の階層仕様整合

- 設計書上の親子階層仕様を、5階層以内のサブタスク階層を正式対応対象とする表現に統一した。
- 完了連動と進捗再計算は直下サブタスク限定ではなく、CHG-045 の正式対応範囲である5階層以内の子孫・祖先を対象とする。

## 2026-05-03 CHG-047 サブタスク管理画面追加

- 詳細ペインに親タスクへのリンクと関連サブタスク一覧を追加し、既存の `parent` frontmatter を使って親子タスク間を遷移できる仕様として起票した。
- DB 変更は行わず、5階層上限仕様との整合を設計・実装・ユーザ確認タスクで追跡する。

## 2026-05-01 CHG-046 ドラッグハンドル並び替え

- メインペインのタスク並び替えは既存の `sort_order` と `tasks:reorder` IPC を利用し、保存モデルは追加しない。
- ドラッグ開始点はタスク行全体ではなく、チェックボックス左隣の `.drag-handle` に限定する。
