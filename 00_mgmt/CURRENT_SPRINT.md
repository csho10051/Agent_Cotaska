### T-015　[完了] タスク進捗ステータス管理

**概要**：タスクごとに「未着 / 仕掛 / 完了」の3段階の進捗ステータスを管理する。既存の todo/doing/blocked/done とは独立した、より粗い粒度の管理軸。

**基本仕様**：

| 項目 | 仕様 |
|------|------|
| DB カラム | `trx_task` に `progress_status TEXT NOT NULL DEFAULT '未着'` を追加 |

| 監査ログ | 記録しない |

* ユーザーが親を手動設定した場合はその値を優先（自動推定で上書きしない）

* 詳細ペイン：progress_status のドロップダウン（未着 / 仕掛 / 完了）
