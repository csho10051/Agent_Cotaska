## 目的

CHG-047 の追加調整として、詳細ペイン下部の「関連サブタスク」が多い場合でも「タスク詳細」本文が押しつぶされないようにする。

## 対応内容

- 詳細ペイン幅をドラッグで変更できることを維持し、変更範囲を明確化する
- 「関連サブタスク」展開時の表示サイズを制限し、件数が多い場合はセクション内スクロールにする
- 画面モックと実装の見た目を揃える

## 完了条件

- 関連サブタスクが多い場合でもタスク詳細本文の表示領域が確保される
- 詳細ペイン幅をドラッグで変更できる
- `npm run build` が成功する

## 実施結果

- 詳細ペイン幅を 320px〜720px の範囲でドラッグ変更できるようにし、変更幅を保存するようにした
- 関連サブタスク一覧に高さ上限と内部スクロールを設定し、タスク詳細本文を押しつぶさないようにした
- 画面モックにもリサイズハンドルと関連サブタスクのスクロール制限を反映した
- `vite build` 成功
