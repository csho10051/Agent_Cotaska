## 概要
期限欄が今はMM/DDであるが、
時分を追加し、MM/DD HH:MMとなるようにしたい。

タスク詳細は、「MM/DD HH:MM」で表示し、
メインパネルのタスクに、
　時分があれば「MM/DD HH:MM」
　時分がなければ「MM/DD」と表示するようにしてほしいです。

あわせて、期限クリック時の編集 UI はシンプルな入力ポップオーバーではなく、
年月日と時刻をまとめて編集できるサブ画面型 UI に見直す。
今回のスコープでは `期日` タブを実装し、`期間` はプレースホルダ扱いとする。

合わせて、進捗ステータスはタスク名の左側に来るようにレイアウト変更したい。
