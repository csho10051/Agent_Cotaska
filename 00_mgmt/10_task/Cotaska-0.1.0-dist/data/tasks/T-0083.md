## 概要

メインペインのタスク一覧で `is_invalid` のタスクを赤系表示・警告アイコン付きで表示する。

## 作業内容

1. タスク行コンポーネントで `is_invalid` を判定する
2. 破損タスクの背景色、左線、警告アイコンなどの表示を追加する
3. 通常タスクと視覚的に区別できるようにする
4. 既存ビュー、フィルター、折りたたみ表示で崩れないことを確認する

## 完了条件

- 破損タスクがタスク一覧に表示される
- 破損タスクが赤系表示・警告アイコンで明確に識別できる

## 対応内容

- `App.jsx` のタスク変換で `is_invalid` と検証エラー情報を renderer へ渡すようにした
- `MainPane.jsx` で破損タスク行に `task-row--invalid` を付与するようにした
- 破損タスクに赤系背景、左線、警告マーク、`要確認` バッジを表示する CSS を追加した
- 破損タスクのチェック、右クリックメニュー、期限変更、ドラッグ操作を無効化した
