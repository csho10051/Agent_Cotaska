# 現象
親タスクを削除した場合、
子タスクの親がなくなって警告表示になってしまう。

# 修正案
子タスクを持つ親タスクを削除する場合は、
警告メッセージ「サブタスクも削除しますか？」を表示する。
⇒「はい」を選択した場合、紐づくタスク全てを削除する。
⇒「いいえ」を選択した場合、親タスクだけを削除し、子タスクの「parent: 」を「null」に更新する。
