## 作業内容

CotaskaにローカルAI CLIチャット連携の共通基盤を実装し、初期プロバイダとしてCodex CLIに対応する。

## 実装方針

- main processにAI CLI実行サービスを追加する。
- CLIごとの差異を吸収するプロバイダ定義を用意する。
- 初期プロバイダとしてCodex CLIを登録する。
- preloadに `cotaskaAPI.aiChat` または同等の名前空間を追加する。
- rendererにチャットペイン、プロバイダ表示、入力欄、応答表示、実行中/エラー状態を追加する。
- CLI応答をチャット表示用メッセージとして扱う。
- タスク更新提案は構造化アクションとして検証し、承認後のみ `taskService` に渡す。

## 完了条件

- [ ] CotaskaからCodex CLIへメッセージを送信できる。
- [ ] CLIプロバイダを追加できる構造になっている。
- [ ] Codex CLI応答がCotaskaのチャット欄に表示される。
- [ ] CLI未検出/未認証/実行失敗がエラーとして表示される。
- [ ] タスク反映提案を承認前プレビューできる。
- [ ] 承認済みアクションのみ既存タスクAPIへ適用される。
