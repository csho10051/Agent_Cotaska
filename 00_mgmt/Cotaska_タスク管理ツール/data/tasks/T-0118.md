# 現象
親タスクを削除した場合、
子タスクの親がなくなって警告表示になってしまう。

# 修正案[廃案]
子タスクを持つ親タスクを削除する場合は、
警告メッセージ「サブタスクも削除しますか？」を表示する。
⇒「はい」を選択した場合、紐づくタスク全てを削除する。
⇒「いいえ」を選択した場合、親タスクだけを削除し、子タスクの「parent: 」を「null」に更新する。

# 追加修正案
子タスクを持つ親タスクを削除する場合は、
警告メッセージ「サブタスクも削除しますか？」を表示する。その際に以下３つのボタンを表示する。
⇒「全て削除」を選択した場合、紐づくタスク全てを削除する。
⇒「親だけ削除」を選択した場合、親タスクだけを削除し、子タスクの「parent: 」を「null」に更新する。
⇒「キャンセル」を選択した場合。削除しない

# 対応結果
アプリ内の確認ダイアログを追加し、「全て削除」「親だけ削除」「キャンセル」の3択で分岐するよう修正した。
