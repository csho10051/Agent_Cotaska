現在バージョンをアプリ内に表示する。

## 対象

- `20_app/package.json`
- `20_app/src/main/`
- `20_app/src/preload/`
- `20_app/src/renderer/`

## 方針

- main process で `package.json` の `version` を取得し、preload 経由で renderer へ渡す。
- renderer は `window.cotaskaAPI` 経由でバージョンを取得し、設定画面のアプリ情報へ表示する。
- T-0021 の設定画面が未完了の場合は、先に API だけ実装できるかを判断する。
