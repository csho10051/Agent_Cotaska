現在バージョンをアプリ内に表示する。

## 対象

- `20_app/package.json`
- `20_app/src/main/`
- `20_app/src/preload/`
- `20_app/src/renderer/`

## 実装結果

- main process に `app:getInfo` IPC を追加し、`package.json` の `version` から `Cotaska 0.1.1` 形式の現在バージョンを返すようにした。
- preload の `window.cotaskaAPI.app.getInfo()` から renderer がアプリ情報を取得できるようにした。
- サイドバーの設定アイコンから設定画面を開けるようにした。
- 設定画面に `アプリ情報` / `設定` / `バックアップ` のサイドパネルを追加した。
- `アプリ情報` では製品名、現在バージョン、更新案内を表示する。
- `設定` では表示名、外部エディタ、通知時間、文字サイズ、外部AIAgent の入力欄を表示する。
- `バックアップ` は未実装表示とした。

## 検証

- `npm run build`
  - システム既定の Node.js 20.18.1 では Vite 要件不足で失敗。
- `v22.14.0` 同梱 Node を PATH 先頭にして `npm.cmd run build`
  - 成功。
- `node --check src/main/main.js`
  - 成功。
- `node --check src/main/preload.js`
  - 成功。
