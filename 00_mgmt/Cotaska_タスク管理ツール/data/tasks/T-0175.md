不具合：プレビューモードでテーブル形式の場合罫線が出ない

## 対応結果

- BUG-20260515-01 としてバグレポートを起票。
- プレビューモードのMarkdownテーブルに罫線・セル余白・ヘッダー背景を追加。
- 同梱Node.js v22.14.0 で `npm.cmd run build` 成功。
- タスク全体検証は既存の T-0169 / T-0170 の日時クォート不備で失敗。T-0175 のfrontmatterは `gray-matter` で読み取り確認済み。
- 2026-05-16 再調査で、ソース修正後の配布物 `app.asar` が未更新だったことを確認。
- `release-all.ps1 -Version "0.1.1"` で `Cotaska-dist` と `Cotaska-dist.zip` を再生成し、配布物内CSSに `.detail-preview table` / `th` / `td` の罫線スタイルが含まれることを確認。
