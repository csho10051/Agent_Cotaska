CotaskaをGitHub Releasesへアップロードしてください。

方針:
- まず release-all.ps1 に Cotaska-dist.zip を作成する処理があるか確認する
- なければ、検証OK後に 20_app/release/Cotaska-dist.zip を作成する処理を追加する
- release-all.ps1 を実行して Cotaska-dist と Cotaska-dist.zip を作成する
- リリース対象の未コミット変更が残っていないか確認する
- 必要な変更があればコミット・pushする
- リリース対象コミットに Gitタグ v0.1.0 を作成する
- Gitタグ v0.1.0 を GitHub へ push する
- GitHub Releases に v0.1.0 のリリースを作成する
- リリースノートは 10_docs/30_実装・検証/20_リリース/20260510_v0.1.0.md を使用する
- Cotaska-dist.zip をリリースアセットとして添付する
- Gitには配布zipや 20_app/release/Cotaska-dist の中身をコミットしない
- 必要な検証を実施し、結果を報告する

補足:
- 既に v0.1.0 タグが存在する場合は、上書きせずに状況を報告して判断を仰ぐ
- リリース対象コミットは、原則として origin/main と同期済みの最新コミットにする
