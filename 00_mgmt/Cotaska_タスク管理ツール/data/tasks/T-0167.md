CHG-060 の実装として、タスク詳細本文の編集欄を Markdown シンタックスハイライト可能なエディタへ置き換える。

## 実装方針

- textarea の疑似ハイライトではなく、CodeMirror 系の Markdown 対応エディタを第一候補とする
- `#`, `##`, `###`, `**強調**`, `*斜体*`, リスト、リンク、コードを編集時に視覚的に区別する
- 保存値は従来どおり `content` の Markdown テキストとする
- プレビューモード、リンク起動、自動保存、無効タスク表示を維持する

## 完了条件

- 編集モードで Markdown 記法がハイライト表示される
- 日本語入力と自動保存が従来どおり動作する
- プレビューモードへ切り替えても既存表示が維持される
- `npm run build` が成功する

## 対応結果

- `DetailPane.jsx` に Markdown 編集ハイライト用のエスケープ済みレンダリングを追加した
- 既存の textarea 入力と 500ms debounce 保存を維持し、背面に Markdown 強調表示を重ねる方式で実装した
- `#`, `##`, `###`, `**強調**`, `*斜体*`, リンク、リスト、引用、インラインコードを視覚的に区別する
- CodeMirror 依存追加は npm install が Google Drive 配下の `node_modules` 書き込みエラーで失敗したため、今回の初期対応では新規依存なしとした
- Codex 同梱 Node v24 を PATH 先頭にして `npm run build` 成功を確認した
