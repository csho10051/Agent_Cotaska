## 目的

Cotaska をデバッグモードで安定して起動できる PowerShell スクリプトを追加する。

## 対応内容

- 同梱 Node.js を利用して Electron を development モードで起動する
- `npx.cmd` が PATH に存在しない環境でも Vite dev server を起動できるようにする
- 既存プロセスが残って画面が出ない状態を避けるため、Cotaska 関連プロセスを整理してから起動できるようにする

## 実施結果

- `20_app/start-debug.ps1` を追加した
- `NODE_ENV=development` / `VITE_PORT` / 同梱 Node.js PATH を設定して起動するようにした
- 既存の Cotaska / Electron / 該当 Node プロセスを停止してから起動する既定動作にした
- `-NoDevTools` 指定時は開発者ツールを自動表示しないよう `main.js` 側も対応した
- PowerShell 構文チェックと Vite build に成功した
