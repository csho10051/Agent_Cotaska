CHG-060 の設計反映として、タスク詳細本文の編集モードに Markdown シンタックスハイライトを追加する仕様を設計書へ記録する。

## 対象

- `10_docs/10_設計/20_基本設計/02_タスク管理.md`
- 必要に応じて `10_docs/10_設計/50_共通資料/実装技術の選定.md`

## 完了条件

- 編集モードで Markdown 記法を強調表示する方針が設計書に反映されている
- 保存データは Markdown テキストのまま維持することが明記されている
- 実装候補として CodeMirror などの Markdown 対応エディタ導入方針が整理されている

## 対応結果

- `10_docs/10_設計/20_基本設計/02_タスク管理.md` の本文編集仕様に CHG-060 を追記した
- 編集モードで Markdown 記法を強調表示すること、保存データは Markdown テキストのまま維持することを明記した
- `10_docs/10_設計/50_共通資料/実装技術の選定.md` は既に CodeMirror 6 をエディタ候補として記載済みのため追加修正なし
