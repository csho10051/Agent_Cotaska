## 概要
DetailPane.jsx を修正し、フッターに登録日時・完了日時を追加する。
メタ情報セクションから「完了日時」表示を削除する。

## 仕様変更票
- CHG-038

## 実装方針
- `DetailPane.jsx` の `detail-footer` に `created_at` の表示を追加
- 完了状態時のみ `completed_at` もフッターに表示
- メタ情報 Row 1 の `{completedAtText && ...}` を削除
- フッター表示形式: `登録日時：YYYY/MM/DD　完了日時：YYYY/MM/DD　ID：T-XXXX`

## 作業記録（2026-03-30）

### 変更ファイル
- `20_app/src/renderer/components/DetailPane.jsx`
  - `formatCompletedAt` を `formatDatetime`（YYYY/MM/DD 形式）にリファクタし、エイリアスとして `formatCompletedAt` を維持
  - `detail-footer` に登録日時スパン（`marginRight: auto` で左端配置）と完了日時スパンを追加
  - メタ情報 Row 1 の `{completedAtText && <span>完了: ...</span>}` を削除
- `npm run build` でビルド成功（✓ 97 modules transformed）
