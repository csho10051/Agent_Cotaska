## 目的

メインペインのクイック追加バーで日付表現を入力したとき、検出された日付を視覚的に強調する。

## 対象仕様変更

- `10_docs/20_実装準備/40_仕様変更管理/20260516_CHG-075_タスク入力補正.md`

## 完了条件

- 日付表現を検出した時点で、クイック追加バー内に日付チップが表示される。
- 日付チップには保存予定の `due_date` が分かる情報を表示する。
- 日付チップは入力欄、リスト候補、リスト選択チップと重ならない。
- 日本語入力中の Enter や既存のタスク作成挙動を壊さない。

## 対応結果

- `20_app/src/renderer/components/MainPane.jsx` で日付検出結果に `dateLabel` を持たせ、入力中の日付表現を日付チップとして表示するようにした。
- `20_app/src/renderer/App.css` に日付チップの強調スタイルを追加した。
- `10_docs/10_設計/20_基本設計/02_タスク管理.md` と `10_docs/10_設計/30_画面モック/layout_mock.html` に日付チップ表示を反映した。
- `npm run build` を Node 22.14.0 環境で実行し、ビルド成功を確認した。
