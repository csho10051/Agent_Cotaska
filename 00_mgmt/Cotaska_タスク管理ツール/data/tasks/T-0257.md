# CHG-078-04 実装：日付ビュー保留セクションに遅延保留を表示

## 目的

今日/明日の日付ビュー内の保留セクションで、遅延している保留タスクも確認できるようにする。

## 作業内容

- 今日/明日/次の7日間ビューで、対象日付範囲に入る保留タスクを日付ビュー内の保留セクションへ表示する。
- 今日ビューでは今日以前、明日ビューでは明日以前、次の7日間ビューでは7日以内の保留タスクを対象にする。
- 未着・仕掛の日付セクションには保留タスクを混ぜない。
- 左ナビの「保留」ビューは従来どおり全保留タスクを表示する。

## 完了条件

- 遅延している保留タスクが今日ビューの保留セクションに表示される。
- 遅延している保留タスクが明日ビューの保留セクションに表示される。
- 保留ではない今日/明日タスクは従来どおり日付セクションに表示される。
- `npm run build` が成功する。

## 実施結果

- `App.jsx` に日付ビュー用の保留タスク抽出処理を追加した。
- 今日 / 明日 / 次の7日間ビューでは、通常の日付セクションに加えて保留セクションも描画できるようにした。
- `MainPane.jsx` で日付セクションと進捗セクションを併用表示できるようにした。
