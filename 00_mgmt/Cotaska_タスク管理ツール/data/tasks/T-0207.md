日付欄の時刻入力が３０分単位でしかできない。
時刻の入力フォーマットはチェックが必要だが、
３０分単位など、入力内容の制限はしたくない。

## 対応記録

- BUG-20260516-01 としてバグレポートを起票した。
- `DueDatePopover.jsx` の手入力バリデーションを修正し、`HH:MM` の分を `00` から `59` まで許可するようにした。
- 同梱 Node.js v22.14.0 で `npm.cmd run build` の成功を確認した。
- 全体タスク検証は既存の T-0169 / T-0170 の日時値クォート不備で失敗したが、T-0207 のfrontmatterは `gray-matter` で正常に読み取り可能であることを確認した。
- 2026-05-16 ユーザー確認でOKを受領した。
