## 目的

Portable版で更新zipを取得し、外部 updater スクリプトによりアプリ終了後に差し替え・再起動する仕組みを実装する。

## 完了条件

- GitHub Releases から `Cotaska-Portable.zip` の asset URL を取得できる。
- 更新zipを一時フォルダへダウンロードできる。
- 外部 updater スクリプトを生成・detached 起動できる。
- updater は `data/`, `logs/`, `backup/` を保持して更新できる。
- 更新前バックアップとエラーログが残る。

## 対応結果

- `updates.check()` で GitHub Releases latest API から `Cotaska-Portable.zip` を検出し、現在バージョンと比較できるようにした。
- `updates.download()` で Portable版 zip を一時フォルダへ保存し、`.sha256` があれば SHA-256、なければ asset size で検証するようにした。
- `updates.install()` で外部 PowerShell updater スクリプトを生成し、detached 起動して Cotaska 終了後に差し替えるようにした。
- updater は `Cotaska.exe`, `_app/`, `tools/`, `README.md`, `Cotaska_AI*.md` を更新対象とし、`data/`, `logs/`, `backup/` は保持する。
- `release-all.ps1` で `Cotaska-Portable.zip.sha256` を生成するようにした。
- `node --check 20_app/src/main/main.js`、`release-all.ps1 -Version "0.2.1"`、`sync-task-master-release.ps1` を実行した。
- 修正版 Portable 起動後、DevTools Protocol 経由で `updates.check()` が `Cotaska-Portable.zip` を検出し、同一バージョンを最新判定することを確認した。
